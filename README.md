WIWU OS V5.1 — Mensal conectado

Alterações:
- Faturamento do dashboard e da aba Faturamento agora é calculado somente por pedidos vinculados a clientes existentes.
- Ao excluir cliente, o sistema remove pedidos, lembretes, histórico e RMA vinculados.
- Aba Faturamento ganhou seletor de mês para consultar meses anteriores/futuros sem criar outro sistema.
- Botão Corrigir faturamento remove pedidos órfãos que ficaram de clientes já apagados.
- Mantém o mesmo visual, Supabase e arquivos separados.

Como subir:
1. Extraia o ZIP.
2. Envie a pasta extraída inteira para o Netlify.
3. Abra o site.
4. Clique em Faturamento > Corrigir para limpar pedidos órfãos antigos, se existirem.
